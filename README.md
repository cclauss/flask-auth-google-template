# Flask auth google template

Flask app with Google auth login

## :floppy_disk: Installation

```bash
python -m venv env
```

```bash
. env/scripts/activate
```

```bash
pip install --upgrade pip
```

```bash
pip install -r requirements.txt
```

## :runner: Run

```bash
flask --debug run --cert=adhoc
```

## :pushpin: Features

- [x] Google Auth login
